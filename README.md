# 洛天依翻唱/原创曲歌词LRC文件

## 项目简介

提供现成的LRC歌词文件

因为是手动对的，所以可能不太准

歌词可能会有点错误，如果歌词明显不准或者有错别字可以发 Issue 告诉我，我会在之后有空时统一改

也可以发 Pull Request

歌词文件没有加密，也可以自己改

格式是[小时:分钟:秒] 歌词

不定时更新

## 歌词滚动
之后可能会加入对 qrc 的支持，现在暂不考虑
